# chat

### Made with TMSSN stack
- **T** = Typescript
- **M** = Mongodb for sure with mongoose
- **S** = Socket.io
- **S** = Svelte
- **N** = Nodejs

### Run dev mode
- Use `cd backend && npm run dev` to run *backend server* in **cmd**
- Use `cd frontend && npm run dev` to run *frontend server* in **cmd**
